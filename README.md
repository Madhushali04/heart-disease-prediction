# Heart Disease Prediction

A machine learning project using the UCI Heart Disease dataset to predict 
the presence of heart disease in patients.

## Dataset
- Source: UCI Machine Learning Repository (ID: 45)
- Features include age, sex, chest pain type, blood pressure, cholesterol, etc.

## Tools & Libraries
- Python
- Pandas, NumPy
- Scikit-learn
- Google Colab
